# Wearable Patient Monitor

Multi-parameter wearable health monitoring device - accepted and presented at 1st International Conference on Medical Device and Drug Delivery Systems 2025, Chennai.

## Tech Stack
- Hardware: Arduino, Temperature Sensor, Pulse Sensor
- Parameters: Heart Rate, SpO2, ECG, Temperature, CO2, Pressure, Accelerometer
- Mobile App: Flutter (Dart)
- Display: Real-time dashboard with status alerts

## Features
- Monitors 7 vital parameters simultaneously
- Wearable form factor for continuous patient monitoring
- Real-time Flutter mobile app dashboard
- Alert system for abnormal readings
- Validated through circuit assembly and system testing
